# Favorite Images — Vencord Plugin

This plugin allows you to mark images as favorites and view them in a dedicated tab.

## How to Install

1. Set up Vencord and its development environment by following the instructions in the [Vencord documentation](https://docs.vencord.dev/installing/)
2. Clone this repository by running `git clone` in your terminal
3. Copy `favoriteImages` folder into `src/userplugins/` inside your Vencord source folder
4. Run `pnpm build` and `pnpm inject`
5. Enable **Favorite Images** in Discord
